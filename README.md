## Developing

1. **Clone the repository:**
  ```sh
  $ git clone git@github.com:gustavomaes/recipe-react.git
  ```
  
2. **Install packages:**
```sh
$ yarn install
```

3. **Run [server](https://github.com/gustavomaes/recipe-server):**
  ```sh
  $ yarn start
  ```
  
4. **Run application:**
```sh
$ yarn start
```
